Placeto CMS Documentation

Welcome! Placeto Content Managment System (CMS) is a website application to create and/or framework almost any website or application off of. It's designed to be simple, semantic, fast, modern, secure, easy to use and light weight. If you're looking to start a brand new website, if your looking to start a website or your even a web development firm, Placeto CMS is for you.
