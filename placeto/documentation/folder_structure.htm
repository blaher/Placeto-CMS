/placeto
  • /admin
    The administator application.
  • /config
    Where all personal config files will be stored.
  • /documentation
    The documentation for Placeto CMS.
  • /engine
    The engine that runs and displays dynamic content. This is mostly application specific code.
  • /library
    All library resource files, containing the class hierarchy. Pretty much framework code.
  • /mods
    Where all modules need to go.
  • /resources
    All common files that are used by Placeto CMS. Like js/css/images.
  • /setup
    The setup system. Use this to install Placeto.
  • /templates
    Where all templates go. Think of this as your views.
