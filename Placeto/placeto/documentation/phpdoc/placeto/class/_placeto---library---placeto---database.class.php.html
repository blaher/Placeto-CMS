Placeto CMS.

A lightweight, easy to use PHP content management system. Written to be as fast as possible and to use as little memory as possible. Placeto provides browser caching, server caching, deflating, and gzip compression, if necessary to cut down on bandwidth and cpu usage.

Class. This is the main abstraction class to fetch all needed data, from the database and other data resources.
